# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ Yes    |

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Email: **hi@yourdomain.com** (replace with your contact)

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive a response within 48 hours. If confirmed, a patched release will be issued and you will be credited in the changelog.
